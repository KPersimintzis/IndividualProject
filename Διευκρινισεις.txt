Individual Project PartA (Private School)

ΔΙΕΥΚΡΙΝΙΣΕΙΣ:

Στο προγραμμα η εισαγωγη των courses καλο ειναι να γινεται στην αρχη είτε προσθετει o user data
ειτε χρησιμοποιουνται τα synthetic.
Αναλυτικοτερα:

Στο user input: 
>Αν δεν εισαγει ο χρηστης αρχικα καποιο course και εισαγει ειτε trainer ειτε student ειτε assignment τα courses 
δημιουργουνται αυτοματα απο τα synthetic.
>Tο ιδιο συμβαινει και σε καθε πεδιο οταν ειναι αδειο και ζηταται να γινει display.Χρησιμοποιουνται τα synthetic

Στο syntheticdata:
>Αν δεν προστεθουν τα courses δεν υπαρχει σωστος συσχετισμος,οποτε μερικες λειτουργιες δεν εμφανιζουν τιποτα.

==================
>Ο χρηστης μπορει να προσθεσει οτι θελει και οποτε θελει ακομα και αν εχουν εισαχθει τα syntheticdata

>To αναποδο ομως δεν γινεται.Απαξ και ο χρηστης κανει μια εισαγωγη(πχ εισαγει εναν νεο student)
 μετα δεν μπορει να προστεθει syntheticdata σε αυτο το πεδιο (πχ εδω στους Students).

==================
>Το output για student,trainer,assignment per course ειναι κοινο.Μπαίνοντας στο μενου 
 courses->display δινεται η δυνατοτητα να εμφανιστουν ολες οι πληροφοριες του εκαστοτε τμηματος.
>Κατι αντιστοιχο γινεται σε ολα τα πεδια.
>Στο students->display υπαρχουν ακομα 2 επιλογες:1.Εμφανιζει τους μαθητες που ειναι εγγεγραμμενοι 
						  σε πανω απο ενα τμημα καθως και τα τμημα τους.
						 2.Αφου δωθει μια ημερομηνια γινεται ελεγχος για το ποιοι
						  μαθητες(και ποιο τμημα) εχουν να παραδωσουν εργασια 
						  μεσα σε εκεινη την εβδομαδα που δόθηκε η ημερομηνια.
